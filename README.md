# Scripting Portfolio

A collection of automation scripts written in **Python**, **Perl**, and **Bash**.

## Scripts

### Python
- `poe_log_analyzer.py` – Game log parser (Path of Exile)
- `file_processor.py` – General CSV/Text file processor
- `system_monitor.py` – System resource monitoring

### Perl
- `email_processor.pl` – Email automation and report processing
- `file_processor_sftp.pl` – File transformation + SFTP upload
- `api_unblock_processor.pl` – API integration with HMAC authentication

### Bash
- `db_backup.sh` – PostgreSQL backup, compression & encryption

## Purpose
Showcasing practical automation, file processing, logging, and system administration skills.

## Technologies
- Python 3
- Perl 5
- Bash
- PostgreSQL, SFTP, GPG, MIME::Lite
